correr script.py para a versão no terminal ou gui.py para a versão com interface gráfica

Os dados são todos públicos no Sigarra (sem fazer login) por incompetência alheia

3 Modos:
1 - Pesquisa por aluno: lista as teses a que um aluno se candidatou
2 - Pesquisa por tese: lista os alunos que se candidataram a uma dada tese
3 - Simulador de colocações: Se apenas a média fosse importante, como seria a distribuição das teses

É necessário fazer o download do excel e da base de dados local para o correto funcionamento do programa, bem como manter na mesma pasta ou editar o código para facilitar o acesso.
